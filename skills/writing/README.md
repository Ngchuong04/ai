# Writing & Documentation

Skills for producing clear prose, technical documentation, illustrations, diagrams, structured data, and professional communication.

## Skills

| Skill | Description |
|-------|-------------|
| [article-illustrator](article-illustrator/) | Analyze article structure and generate illustrations using a Type x Style two-dimension approach. |
| [brainstorming](brainstorming/) | Explore ideas before implementation through collaborative dialogue that turns ideas into designs and specs. |
| [clear-writing](clear-writing/) | Write clear, concise prose for documentation, READMEs, API docs, commit messages, and technical writing. |
| [game-changing-features](game-changing-features/) | Find 10x product opportunities and high-leverage improvements through strategic product thinking. |
| [mermaid-diagrams](mermaid-diagrams/) | Create software diagrams using Mermaid syntax — flowcharts, sequence diagrams, ERDs, C4, and more. |
| [persona-docs](persona-docs/) | Create persona documentation that defines target users, journeys, and onboarding flows for a product. |
| [professional-communication](professional-communication/) | Write effective professional messages for software teams — emails, Slack, meeting agendas, and status updates. |
| [prompt-engineering](prompt-engineering/) | Master advanced prompt engineering techniques to maximize LLM performance, reliability, and controllability. |
| [schema-markup](schema-markup/) | Add, fix, or optimize schema.org markup and structured data for rich search results. |
