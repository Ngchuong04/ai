# AI Chat Application

Complete guide to building AI chat applications with Vercel AI SDK. Orchestrates streaming routes, useChat integration, tool architecture, message handling, and UX patterns for production AI chatbots and assistants.

## What's Inside

- Architecture overview (client, API route, tool system, message handling, AI persona)
- Step-by-step implementation (API route, chat component, UX polish, tool system, message persistence, AI persona)
- Component skills reference table
- Quick start checklist

## When to Use

- Building a new AI chatbot or assistant
- Adding AI chat features to an existing app
- Need end-to-end guidance on AI chat architecture

## Installation

```bash
npx skills add ai-chat-application
```

### Manual Installation

#### Cursor (per-project)

From your project root:

```bash
mkdir -p .cursor/skills
cp -r ~/.ai-skills/skills/meta/ai-chat-application .cursor/skills/ai-chat-application
```

#### Cursor (global)

```bash
mkdir -p ~/.cursor/skills
cp -r ~/.ai-skills/skills/meta/ai-chat-application ~/.cursor/skills/ai-chat-application
```
