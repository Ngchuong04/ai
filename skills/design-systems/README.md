# Design Systems

Skills for building scalable design system architecture — token hierarchies, theming infrastructure, component libraries, visual identity, and data visualization patterns.

## Skills

| Skill | Description |
|-------|-------------|
| [animated-financial-display](animated-financial-display/) | Patterns for animating financial numbers with spring physics, formatting, and visual feedback. |
| [design-system-components](design-system-components/) | Build reusable UI components using Surface primitives, CVA variants, and design token architecture. |
| [design-system-patterns](design-system-patterns/) | Foundational design system architecture — token hierarchies, theming infrastructure, token pipelines, and governance. |
| [distinctive-design-systems](distinctive-design-systems/) | Create design systems with personality and distinctive aesthetics beyond generic templates. |
| [financial-data-visualization](financial-data-visualization/) | Build dark-themed financial charts and data visualizations for trading dashboards and analytics. |
| [loading-state-patterns](loading-state-patterns/) | Patterns for skeleton loaders, shimmer effects, and loading states that match design system aesthetics. |
| [theme-factory](theme-factory/) | Curated collection of professional color and typography themes for styling slides, docs, and reports. |
| [ui-design](ui-design/) | Comprehensive UI design fundamentals covering layout, typography, color, spacing, accessibility, and motion. |
| [ui-ux-pro-max](ui-ux-pro-max/) | Searchable UI/UX design databases with CLI for generating complete design systems from natural language. |
| [web-design](web-design/) | CSS implementation patterns for layout, typography, color, spacing, and responsive design. |

## Installation

```bash
# Add individual skills
skills add animated-financial-display
skills add design-system-patterns
skills add theme-factory
```
