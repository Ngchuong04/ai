# Infrastructure

Skills for containerization, orchestration, and observability — the foundational infrastructure patterns that make applications deployable, scalable, and observable in production. This category covers Docker image optimization, Kubernetes manifest generation, and Prometheus monitoring configuration with production-grade security, health checks, and operational best practices.

## Skills

| Skill | Description |
|-------|-------------|
| [docker-environment-patterns](docker-environment-patterns/) | Docker containerization expertise — multi-stage builds, image optimization, security hardening, Docker Compose orchestration, and production deployment patterns. |
| [kubernetes-deployment-patterns](kubernetes-deployment-patterns/) | Production-ready Kubernetes manifest generation covering Deployments, StatefulSets, CronJobs, Services, Ingresses, ConfigMaps, Secrets, and PVCs with security contexts and health checks. |
| [prometheus-observability](prometheus-observability/) | Prometheus monitoring — scrape configuration, service discovery, recording rules, alert rules, and production deployment for infrastructure and application metrics. |

## Installation

```bash
skills add docker-environment-patterns
skills add kubernetes-deployment-patterns
skills add prometheus-observability
```
