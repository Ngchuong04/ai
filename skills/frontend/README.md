# Frontend

Skills for frontend development — React component patterns, performance optimization, mobile native UI, Next.js, design systems with Tailwind CSS, and responsive design.

## Skills

| Skill | Description |
|-------|-------------|
| [composition-patterns](composition-patterns/) | React composition patterns using compound components, context providers, and explicit variants. |
| [expo-native-ui](expo-native-ui/) | Build native iOS/Android apps with Expo Router, native tabs, animations, and platform patterns. |
| [frontend-design](frontend-design/) | Create distinctive, production-grade frontend interfaces that avoid generic AI-generated aesthetics. |
| [native-ui](native-ui/) | Build native mobile UIs with Expo Router and React Native following Apple Human Interface Guidelines. |
| [nextjs](nextjs/) | Next.js App Router best practices — Server Components, data fetching, caching, routing, and performance. |
| [react-best-practices](react-best-practices/) | React and Next.js performance optimization guidelines with 57 rules across 8 categories from Vercel Engineering. |
| [react-composition](react-composition/) | React composition patterns for scalable component architecture with compound components and context providers. |
| [react-modernization](react-modernization/) | Upgrade React applications from class components to hooks and adopt concurrent features. |
| [react-performance](react-performance/) | React and Next.js performance optimization patterns for components, data fetching, and bundle optimization. |
| [responsive-design](responsive-design/) | Modern responsive CSS patterns using container queries, fluid typography, CSS Grid, and mobile-first strategies. |
| [shadcn-ui](shadcn-ui/) | Build accessible, customizable UIs with shadcn/ui, Radix UI, and Tailwind CSS. |
| [tailwind-design-system](tailwind-design-system/) | Build production-ready component libraries with Tailwind CSS using CVA, compound components, and theming. |
| [tailwind-v4-shadcn](tailwind-v4-shadcn/) | Production-tested setup for Tailwind v4 with shadcn/ui preventing documented errors through mandatory architecture. |
