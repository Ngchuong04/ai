# Copywriting

Write clear, compelling marketing copy that drives action. Covers homepage, landing pages, pricing pages, feature pages, about pages, and product pages.

## What's Inside

- Workflow — gather context, write copy, provide alternatives, include meta content
- Core Principles — clarity over cleverness, benefits over features, specificity over vagueness
- Writing Style Rules — simple, specific, active, confident, honest
- Headline Formulas — outcome-focused, problem-focused, audience-focused, proof-focused
- CTA Guidelines — strong vs weak CTAs with formulas
- Page-Specific Guidance — homepage, landing page, pricing, feature, about
- Quality Checklist — pre-publish copy review
- Reference: Copy Frameworks — headline formulas, page templates, section types
- Reference: Natural Transitions — transitional phrases and signposting

## When to Use

- Writing, rewriting, or improving marketing copy
- Creating copy for homepage, landing pages, pricing pages, feature pages, or about pages
- Crafting headlines, CTAs, or value propositions
- Reviewing existing copy for clarity and conversion potential
