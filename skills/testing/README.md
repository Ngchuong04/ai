# Testing & Code Quality

Skills for writing clean code, reviewing changes, debugging issues, and building comprehensive test suites — from unit tests through end-to-end coverage and quality gates.

## Skills

| Skill | Description |
|-------|-------------|
| [clean-code](clean-code/) | Pragmatic coding standards for writing clean, maintainable code — naming, functions, structure, and anti-patterns. |
| [code-review](code-review/) | Systematic code review patterns covering security, performance, maintainability, correctness, and testing. |
| [debugging](debugging/) | Systematic debugging approaches using the scientific method applied to code. |
| [e2e-testing-patterns](e2e-testing-patterns/) | Build reliable, fast E2E test suites with Playwright and Cypress. |
| [quality-gates](quality-gates/) | Quality checkpoints at every development stage from pre-commit through post-deploy. |
| [reducing-entropy](reducing-entropy/) | Minimize total codebase size through ruthless simplification and bias toward deletion. |
| [testing-patterns](testing-patterns/) | Unit, integration, and E2E testing patterns with framework-specific guidance. |
| [testing-workflow](testing-workflow/) | Meta-skill that orchestrates comprehensive testing across a project by coordinating testing skills and agents. |

## Installation

```bash
# Add individual skills
npx skills add clean-code
npx skills add code-review
npx skills add debugging
npx skills add e2e-testing-patterns
npx skills add quality-gates
npx skills add reducing-entropy
npx skills add testing-patterns
npx skills add testing-workflow
```
