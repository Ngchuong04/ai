# Marketing Agent

Autonomous workflow for creating and optimizing marketing pages, copy, and conversion funnels — from audience research through copy creation, CRO analysis, and measurement.

## Workflow Phases

- **Phase 1: Audience research** — Target personas, pain points, jobs-to-be-done, buyer's journey
- **Phase 2: Strategy** — Channel strategy, conversion funnel architecture
- **Phase 3: Copy creation** — Headlines, CTAs, body copy using PAS, AIDA, StoryBrand, psychology
- **Phase 4: Page design** — Landing pages, signup flows, onboarding sequences
- **Phase 5: CRO analysis** — Audit pages and flows for conversion optimization
- **Phase 6: Measurement** — KPIs, measurement plans, A/B testing recommendations

## Skills Used

- `copywriting` — Marketing copy frameworks, headlines, CTAs
- `page-cro` — Page conversion optimization audits
- `signup-flow-cro` — Signup flow optimization
- `onboarding-cro` — Post-signup onboarding optimization
- `marketing-psychology` — Mental models for persuasion
- `marketing-ideas` — Proven marketing strategies
- `social-content` — Platform-specific social content
- `content-strategy` — Content strategy and planning
- `startup-metrics` — KPIs, unit economics, tracking
- `persona-docs` — Target persona documentation
